## ChatGPT powered assistant guide

## Model details: Gpt-4o 

Currently using GPT-4o for the chatbot through Github Copilot. Since the model is a tuned version of GPT-4, it is also integrated with Visual Studio Code, and it doesn't have a strict limit on the number of tokens.

## Prompting Guides

Prompting guides for GPT-4o can be found at [Prompt Engineering](https://platform.openai.com/docs/guides/prompt-engineering). 

Similar to MetaAI, GPT-4o does not have any special formatting to best understand context.
